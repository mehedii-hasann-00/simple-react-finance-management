# FinEase - Personal Finance Management

FinEase is a personal finance management web application that helps users manage their income, expenses, and savings goals. It allows users to record transactions, set monthly budgets, and view financial summaries with insightful charts and reports.

## Live Site

You can access the live site here: [FinEase Live Site](https://legendary-torte-c7f6d8.netlify.app/)

## Features

- **User Authentication**: Allows users to log in via email/password or Google login.
- **Transaction Management**: Users can add, view, update, and delete transactions.
- **Financial Overview**: Displays total balance, income, and expenses.
- **Reports & Analytics**: Generates financial reports with charts (e.g., pie charts for categories, bar charts for monthly totals).
- **Responsive Design**: Mobile-friendly layout ensuring accessibility across devices.
- **Profile Management**: Users can update their profile and view their transaction history.
- **Dark Mode**: Toggle between light and dark modes for a better user experience.
- **Error Handling**: All error and success messages are shown using SweetAlert/Toast notifications.
  
## Technologies Used

- **Frontend**:
  - React.js
  - Tailwind CSS
  - React Router
  - Chart.js (for reports)

  
- **Backend**:
  - Node.js
  - Express.js
  - MongoDB
  - Firebase Authentication
